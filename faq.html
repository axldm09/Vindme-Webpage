Frequently Asked Questions

Free Trial

Yes, we offer a 7-day free trial for our Premium Plan. This allows you to experience all the features and benefits of our app before committing to a subscription.

If you choose not to subscribe after the free trial period, your account will automatically revert to our Free Plan. You'll still have access to basic features, but some premium features will be restricted.

Set Up

Our app is designed to be incredibly user-friendly and easy to set up. Simply download the app, create an account, and follow the guided setup process. You'll be up and running in minutes.

No technical expertise is required. Our intuitive interface and step-by-step instructions make it easy for anyone to set up and use the app.

Friends

To add a friend, simply navigate to the main panel and click on the "Add Member" button. Enter their phone number, including the country code if they are from a different country, and send them an invitation. Please note that this feature is available exclusively for Premium Plan subscribers.

Yes, your friend will receive a notification informing them that you've added them on the app. They'll have the option to accept or decline your invitation.

The ability to add multiple friends at once is a premium feature available exclusively for Premium Plan subscribers. Simply import your contacts or manually enter their information, ensuring each contact has a valid phone number with the country code included.

Yes, to add a contact, they must have a valid phone number and if they are outside the country the phone number must include the country code (e.g., +1 for the United States). This ensures accurate and seamless connection within the app.
